# FumoUtils (Public)
Public edition of FumoUtils.

### Requirements
- Minecraft 1.21
- Fabric loader
- Meteor Client

### Available modules:
- AltitudeStabilizer
- AutoDump
- ChatNotifier
- DoujinDupe
- ElytraWatch
- FumoVoidESP
- HighwayHighlighter
- ItemESP
- MapHighlighter
- QuartzFarmer
- RegionFileHighlighter
- UnSilkToucher
- WebhookBridge
- WireproxyIntegration

### Available commands:
- ChatCooker
- OfflineUUIDGen
- PrintTPS
