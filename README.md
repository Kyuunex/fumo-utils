# FumoUtils (Public)
Public edition of FumoUtils.

### Requirements
- Minecraft 1.21.1
- Fabric loader
- Meteor Client

### Available modules:
- AutoDump
- ChatNotifier
- DoujinDupe
- ElytraWatch
- HighwayHighlighter
- ItemESP
- MapHighlighter
- QuartzFarmer
- RegionFileHighlighter
- UnSilkToucher
- WebhookBridge

### Available commands:
- ChatCooker
- OfflineUUIDGen
- PrintTPS
